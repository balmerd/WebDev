Something about the Revealing Module pattern...

See: http://stackoverflow.com/questions/5647258/how-to-use-revealing-module-pattern-in-javascript

"You put everything in a closure (function) and expose only those part that you wish to be accessible. The difference between Module style (MP) and Revealing Module (RMP) is that in MP functions are defined in different places while in the RMP they're always defined in the same place and then afterwards revealed in the public object literal."

Code:
var obj = (function() {
	var privateProperty = 2;
	var privateMethod = function() {
		console.log('private');
	};
	var myExposedMethod = function() {
		console.log('exposed!');
	};
	return  {
		PublicMethod: myExposedMethod
	};

}()); // execute the anonymous function a.k.a. "self-instantiation"

obj.PublicMethod();
Console Output:
exposed!
